### Microsoft Excel Login

Auto Clicker Auto Fill needs read-only access to your Microsoft Excel workbooks to fetch cell values. Click `Connect with Microsoft Excel` and grant the following scopes.

### Microsoft Excel Scope

- `https://graph.microsoft.com/Files.Read`
- `https://graph.microsoft.com/User.Read`

> After you connect with Microsoft Excel go to configuration settings and add [Microsoft Excel Workbook ID]([[docsref:/configuration/config-settings#microsoft-excel-workbook-id]])
