<BsTable>

| **Attr::**             | **Description**                                                                                                                                                                             |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Attr::set::prop::value | Update attribute value. e.g. Attr::set::disabled::false &lt;input disabled="true"&gt; -> &lt;input disabled="false"&gt;; Attr::set::target::\_blank &lt;a&gt; -> &lt;a target="\_blank"&gt; |
| Attr::remove::prop     | Remove attribute. e.g. Attr::remove::disabled &lt;input disabled="true"&gt; -> &lt;input&gt;                                                                                                |

</BsTable>
