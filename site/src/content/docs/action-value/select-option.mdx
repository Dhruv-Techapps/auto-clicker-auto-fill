### Example

<BsTable>

| **Element Finder**                                               | **Selection**                                   |
| ---------------------------------------------------------------- | ----------------------------------------------- |
| `//select[@id="product-size"]/option[2]`                         | Selects the second option with text 35.         |
| `//select[@id="product-size"]/option[@data-value-size="36"]`     | Selects the option with data-value-size 36.     |
| `//select[@id="product-size"]/option[contains(@value,"opt-40")]` | Selects the option whose value contains opt-40. |
| `//select[@id="product-size"]/option[text()="39"]`               | Selects the option with text 39.                |

</BsTable>

<Callout type="info">You must set the value field to `true`</Callout>
