e.g. In value field if you enter `example<batchRepeat>`
e.g. If batch repeat is `5` it will run till `example4`

## Supported Patterns

<BsTable>

| **Pattern**                       | **Description**                                                  |
| --------------------------------- | ---------------------------------------------------------------- |
| [`<batchRepeat>`](#batchrepeat)   | Replaces with the current batch repeat index (starting from 0).  |
| [`<actionRepeat>`](#actionrepeat) | Replaces with the current action repeat index (starting from 0). |
| [`<sessionCount>`](#sessioncount) | Replaces with the current session count (starting from 0).       |

</BsTable>

### `<batchRepeat>`

`example<batchRepeat>`

### `<actionRepeat>`

`item<actionRepeat>`
