tags: [batch, batch-repeat]

---

e.g. In value field if you enter `example<batchRepeat>`
e.g. If batch repeat is `5` it will run till `example4`

## Supported Patterns

<table class="table">
  <thead>
    <tr>
      <th scope="col">Pattern</th>
      <th scope="col">Description</th>
      <th scope="col">Example</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">&lt;batchRepeat&gt;</th>
      <td>Replaces with the current batch repeat index (starting from 0).</td>
      <td>
        <code>example&lt;batchRepeat&gt;</code> → <code>example0</code>, <code>example1</code>, ...
      </td>
    </tr>
    <tr>
      <th scope="row">&lt;actionRepeat&gt;</th>
      <td>Replaces with the current action repeat index (starting from 0).</td>
      <td>
        <code>item&lt;actionRepeat&gt;</code> → <code>item0</code>, <code>item1</code>, ...
      </td>
    </tr>
    <tr>
      <th scope="row">&lt;sessionCount&gt;</th>
      <td>Replaces with the current session count (starting from 0).</td>
      <td>
        <code>run&lt;sessionCount&gt;</code> → <code>run0</code>, <code>run1</code>, ...
      </td>
    </tr>
  </tbody>
</table>
