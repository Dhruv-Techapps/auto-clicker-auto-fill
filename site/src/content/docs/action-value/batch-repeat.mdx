e.g. In value field if you enter `example<batchRepeat>`
e.g. If batch repeat is `5` it will run till `example4`

## Supported Patterns

<BsTable>

| **Pattern**          | **Description**                                                  | **Example**                                          |
| -------------------- | ---------------------------------------------------------------- | ---------------------------------------------------- |
| &lt;batchRepeat&gt;  | Replaces with the current batch repeat index (starting from 0).  | example&lt;batchRepeat&gt; → example0, example1, ... |
| &lt;actionRepeat&gt; | Replaces with the current action repeat index (starting from 0). | item&lt;actionRepeat&gt; → item0, item1, ...         |
| &lt;sessionCount&gt; | Replaces with the current session count (starting from 0).       | run&lt;sessionCount&gt; → run0, run1, ...            |

</BsTable>
