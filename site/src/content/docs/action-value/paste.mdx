<BsTable>

| **Paste::**                          | **Description**                                    |
| ------------------------------------ | -------------------------------------------------- |
| `Paste::`                            | Paste the content copied using the copy command.   |
| `<paste>`                            | Placeholder that expands to the last copied value. |
| `Paste::at(5)`                       | Return character at position 5.                    |
| `Paste::charAt(4)`                   | Same as at(4).                                     |
| `Paste::concat("World")`             | Concatenate strings.                               |
| `Paste::match(/[A-Z]/g)`             | Regex match.                                       |
| `Paste::replace("dog", "monkey")`    | Replace first occurrence.                          |
| `Paste::replaceAll("dog", "monkey")` | Replace all occurrences.                           |
| `Paste::slice(31)`                   | Slice from index 31.                               |
| `Paste::split(" ")`                  | Split by space.                                    |
| `Paste::substring(1, 3)`             | Substring from 1 to 3 (exclusive).                 |
| `Paste::toLowerCase()`               | Lowercase.                                         |
| `Paste::toUpperCase()`               | Uppercase.                                         |

</BsTable>
