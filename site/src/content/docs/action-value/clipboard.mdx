<BsTable>

| **Clipboard::Copy::**     | **Description (Copy text from webpage) to clipboard** |
| ------------------------- | ----------------------------------------------------- |
| Clipboard::Copy::[.]{6}   | `.` Any single character; `{6}` exactly 6 characters. |
| Clipboard::Copy::[.]+     | `.` Any single character; `+` one or more characters. |
| Clipboard::Copy::[a-z]{6} | `[a-z]` a character in range a-z.                     |
| Clipboard::Copy::[A-Z]{6} | `[A-Z]` a character in range A-Z.                     |
| Clipboard::Copy::[\\d]{6} | `[\\d]` a digit.                                      |
| Clipboard::Copy::[\\w]{6} | `[\\w]` a word character.                             |

</BsTable>

<BsTable>

| **Pattern** | **Description** |
|-------------|------------------|
| &lt;clipboardPaste&gt; | Replaces with the current clipboard content. You can use `&lt;clipboardPaste&gt;` anywhere within your text, and it will be replaced with the clipboard value. Supports multiple replacements in the same string. Example: `Hello &lt;clipboardPaste&gt; World &lt;clipboardPaste&gt;!` → `Hello copiedText World copiedText!` |
| Clipboard::Paste:: | Paste the current clipboard content or the value copied using Clipboard::copy::. |
| Clipboard::Paste::at(5) | Character at position 5. Example: `'The quick brown fox jumps over the lazy dog.'.at(5)` → `u`; at(-4) → `d`. |
| Clipboard::Paste::charAt(4) | Same as at(4). Example: `'The quick brown fox jumps over the lazy dog.'.charAt(4)` → `q`. |
| Clipboard::Paste::concat("World") | Concatenate strings. Example: `'Hello'.concat(' ', 'World')` → `Hello World`; with comma → `Hello, World`. |
| Clipboard::Paste::match(/[A-Z]/g) | Regex match. Example: `'The quick brown fox jumps. It barked.'.match(/[A-Z]/g)` → `["T","I"]`; `.join("")` → `TI`; `.join(",")` → `T,I`. |
| Clipboard::Paste::replace("dog", "monkey") | Replace first occurrence. |
| Clipboard::Paste::replaceAll("dog", "monkey") | Replace all occurrences. |
| Clipboard::Paste::slice(31) | Slice examples: `slice(31)` → `the lazy dog.`; `slice(4,19)` → `quick brown fox`; `slice(-4)` → `dog`; `slice(-9,-5)` → `lazy `. |
| Clipboard::Paste::split(" ") | Split by space; examples: `split(' ')[3]` → `fox`, `split('')[8]` → `k`. |
| Clipboard::Paste::substring(1, 3) | Substring from 1 to 3 (exclusive); `substring(5)` → `Clicker AutoFill`. |
| Clipboard::Paste::toLowerCase() | Lowercase. Example: `auto clicker autofill`. |
| Clipboard::Paste::toUpperCase() | Uppercase. Example: `AUTO CLICKER AUTOFILL`. |
| Clipboard::Paste::trim() | Trim both ends. Example: `'     Auto Clicker AutoFill     '.trim()` → `Auto Clicker AutoFill`. |
| Clipboard::Paste::trimStart() | Trim start only. |
| Clipboard::Paste::trimEnd() | Trim end only. |

</BsTable>
