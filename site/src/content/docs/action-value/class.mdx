<BsTable>

| **Class::**                                                                               | **Description**                      |
| ----------------------------------------------------------------------------------------- | ------------------------------------ |
| [Class::add::className](#classaddclassname)                                               | Add a class to the element.          |
| [Class::remove::className](#classremoveclassname)                                         | Remove a class from the element.     |
| [Class::replace::removeClassName::addClassName](#classreplaceremoveclassnameaddclassname) | Replace one class with another.      |
| [Class::clear](#classclear)                                                               | Remove all classes from the element. |

</BsTable>

## Class::add::className

<Code code={`<input class="button">  → <input class="button btn">`} lang="html" />

## Class::remove::className

<Code code={`<input class="button btn">  → <input class="button">`} lang="html" />

## Class::replace::removeClassName::addClassName

<Code code={`<input class="button btn">  → <input class="button like">`} lang="html" />

## Class::clear

<Code code={`<input class="button btn">  → <input class="">`} lang="html" />
