<BsTable>

| **Class::**                                   | **Description**                                                                                             |
| --------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| Class::add::className                         | Add a class to the element. Example: `<input class="button">` → `<input class="button btn">`                |
| Class::remove::className                      | Remove a class from the element. Example: `<input class="button btn">` → `<input class="button">`           |
| Class::replace::removeClassName::addClassName | Replace one class with another. Example: `<span class="unlike">` → `<span class="like">`                    |
| Class::clear                                  | Remove all classes from the element. Example: `<input class="btn btn-primary button">` → `<input class="">` |

</BsTable>
