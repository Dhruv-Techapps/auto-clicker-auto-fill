  variables={{

    value: "KeyEvents::{'value':'Example text','delay':3}"
  }}
/>

<BsTable>

| **KeyEvents::**                                 | **Description**                                              |
| ----------------------------------------------- | ------------------------------------------------------------ |
| `KeyEvents::Example Text`                       | Simulates typing of “Example Text” like real keyboard input. |
| `KeyEvents::{"value":"Example text","delay":3}` | Simulates typing with a 3s delay between each character.     |

</BsTable>
