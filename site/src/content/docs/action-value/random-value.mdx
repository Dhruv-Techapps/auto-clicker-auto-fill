> By Default the length of string is fixed to 6 and filter is set to all character + numbers + symbol so when you use `<random>` you will get random string of length 6 with mix of char + num + symbol

<BsTable>

| **&lt;random&gt;**           | **Description (Generate Random value)**                    |
| ---------------------------- | ---------------------------------------------------------- |
| &lt;random[.]\{6\}&gt;       | [.] any character; {6} exactly 6 characters                |
| &lt;random[.]\{(6, 12)\}&gt; | [.] any character; {(6, 12)} 6 to 12 characters            |
| &lt;random[a-z]\{6\}&gt;     | [a-z] a character in range a–z; {6} length 6               |
| &lt;random[A-Z]\{6\}&gt;     | [A-Z] a character in range A–Z; {6} length 6               |
| &lt;random[\d]\{6\}&gt;      | [\d] a digit; {6} length 6                                 |
| &lt;random[\w]\{6\}&gt;      | [\w] a word character; {6} length 6                        |
| &lt;random[custom]\{6\}&gt;  | [custom] custom character set (e.g., csuotm); {6} length 6 |

</BsTable>
